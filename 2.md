LÓGICA DE BENEFICIARIOS CUANDO NO ES ESPOSA
CONTEXTO

Este flujo aplica cuando el cliente dice:

falleció mi hermano
falleció mi padre
falleció mi padrastro
falleció un familiar DISTINTO a pareja

FLUJO DEL BOT
🔹 PASO 1: DETECTAR CASO

Si el cliente NO es esposa, compañera o pareja

👉 activar flujo de validación de beneficiarios

🔹 PASO 2: PRIMERA PREGUNTA (OBLIGATORIA)

Mensaje:

¿El fallecido dejó esposa, compañera o hijos menores de edad?

🔹 PASO 3: RESPUESTAS
✔ SI RESPONDE SÍ (hay esposa o hijos menores)

👉 El bot NO continúa con más preguntas
👉 Debe pasar directamente a:

Mensaje 1:
Entiendo

Mensaje 2:
Para consultar el caso, por favor envíame:

Cédula del fallecido
Fecha exacta de fallecimiento (día, mes y año)
❌ SI RESPONDE NO

👉 Continuar al siguiente nivel

🔹 PASO 4: SEGUNDA PREGUNTA

Mensaje:

¿El fallecido dejó padres con vida?

✔ SI RESPONDE SÍ

👉 Solicitar datos

Mensaje 1:
Perfecto

Mensaje 2:
Para consultar el caso, envíame:

Cédula del fallecido
Fecha exacta de fallecimiento
❌ SI RESPONDE NO

👉 Continuar

🔹 PASO 5: TERCERA PREGUNTA (FINAL)

Mensaje:

¿El fallecido dejó algún hijo o hermano con discapacidad que dependiera económicamente de él?

✔ SI RESPONDE SÍ

👉 Solicitar datos

Mensaje 1:
Entiendo

Mensaje 2:
Para revisar el caso, envíame:

Cédula del fallecido
Fecha exacta de fallecimiento
❌ SI RESPONDE NO

👉 CERRAR CASO

Mensaje:

Desafortunadamente, en este caso no existen beneficiarios directos para reclamar una pensión de sobrevivientes.

REGLAS IMPORTANTES PARA EL BOT
❗ No saltarse el orden
(esposa/hijos → padres → discapacitados)
❗ No hacer todas las preguntas juntas
(una por una)
❗ No pedir datos hasta confirmar que hay beneficiario
❗ Mensajes cortos (1–2 líneas)
❗ No repetir preguntas
EJEMPLO RÁPIDO

Cliente:
Murió mi hermano

Bot:
¿Dejó esposa, compañera o hijos menores?

Cliente:
No

Bot:
¿Dejó padres con vida?

Cliente:
No

Bot:
¿Dejó algún familiar con discapacidad que dependiera de él?

Cliente:
No

Bot:
Desafortunadamente no existen beneficiarios directos para reclamar pensión en este caso.
